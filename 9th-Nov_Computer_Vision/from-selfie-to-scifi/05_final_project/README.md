# Sci-Fi Filter - Final Project

This project combines all the previous modules to create an immersive sci-fi filter effect.

## Features

- Face detection
- Facial landmark detection
- Filter overlay
- Emotion-based glow effects
- Sci-fi themed visual effects

## Assets

Place your sci-fi filter assets in the `assets/` folder:
- `visor.png` - Sci-fi visor overlay
- `neon_ring.png` - Neon ring effect
- Additional assets as needed

## Usage

```python
from sci_fi_filter import apply_sci_fi_filter

# Apply filter to an image
result = apply_sci_fi_filter('input.jpg', 'output.jpg')
```
